xtpxlib-webdoc is part of the xtpxlib library. xtpxlib contains software for processing XML, using languages like XSLT and XProc. It consists of several separate components, named xtpxlib-*. Everything can be found on GitHub (https://github.com/xatapult). The core component of xtpxlib is xtpxlib-common. Most components will be accompanied by a GitHub pages documentation site (TBD).
